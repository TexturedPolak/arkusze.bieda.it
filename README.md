# Arkusze dla techników informatyków
Strona zawiera zbiór arkuszy CKE egzaminów INF.02, INF.03, EE.08, EE.09 z poprzednich lat, oraz ewentualne pliki do nich.
Pochodzą głównie z [egzamin.it](https://egzamin.it).
## Jak korzystać?
Arkusze są posegregowane kolejno: Podstawa Programowa > Specyfikacja > Rok > styczeń/czerwiec > numer arkusza > arkusz
## Jak sprawnie pobrać wszystko?
Znajdź link "zip" znajdujący się na lewo od nazwy folderu i go klliknij. Pobierze się zip ze wszystkim.
## Jakie jest hasło do zipa przypisanego do arkusza?
Hasło **zawsze jest w arkuszu**.
